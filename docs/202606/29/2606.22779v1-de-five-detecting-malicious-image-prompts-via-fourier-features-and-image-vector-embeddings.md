## 摘要
视觉语言模型（VLMs）利用视觉和文本模态实现高级视觉语言推理。然而，引入视觉模态扩大了VLM的攻击面，使其更易受到对抗扰动和间接提示注入等安全威胁，其中精心设计的恶意图像提示可能引发非预期的模型输出。现有的针对恶意图像提示的防御方法仍显不足，因为它们通常需要大量数据集进行重新训练或部署额外的复杂分类器。最关键的是，目前严重缺乏专门针对间接提示注入的防御机制，这一空白是本研究的主要动机。为应对这些局限，我们提出了DE-FIVE，一种新颖的免训练框架，通过利用傅里叶特征和视觉编码器在不同扰动下的隐藏状态表示（图像向量嵌入）来检测恶意图像提示。具体而言，我们开发了一种混合检测策略，包括一个基于傅里叶域特征运作的黑盒检测器和一个利用仅从少量恶意样本集导出的图像向量嵌入的白盒检测器。大量实验表明，所提出的框架在防御恶意图像提示方面始终优于最先进的基线方法。

## Abstract
Vision language models (VLMs) employ both visual and textual modalities to enable advanced vision-language inference. However, incorporating visual modalities expands the attack surface of VLMs, making them more susceptible to security threats such as adversarial perturbations and indirect prompt injection, wherein crafted malicious image prompts can elicit unintended model outputs. Existing defense methods against malicious image prompts remain insufficient as they typically demand extensive datasets for retraining or the deployment of additional, complex classifiers. Most critically, there is a profound lack of specialized defense mechanisms specifically targeting indirect prompt injections, a gap that serves as a primary motivation for this work. To address these limitations, we introduce DE-FIVE, a novel training-free framework for detecting malicious image prompts by leveraging Fourier features and the hidden state representations of the visual encoder (image vector embeddings) across perturbations. Specifically, we develop a hybrid detection strategy consisting of a black-box detector that operates on Fourier-domain features and a white-box detector that exploits image vector embeddings derived from only a few-shot malicious set. Extensive experiments demonstrate that the proposed framework consistently outperforms state-of-the-art baselines against malicious image prompts.

### 1. 核心问题与整体含义
视觉语言模型（VLMs）将视觉模态与文本模态融合，显著提升了多模态推理能力，但也因此扩大了攻击面，易受**恶意图像提示**攻击。这类攻击主要分为两类：  
- **越狱攻击（jailbreak）**：用户自身充当攻击者，通过精心设计的图像扰动绕过模型安全对齐。  
- **间接提示注入（indirect prompt injection）**：第三方将恶意指令隐式嵌入图像，受害者在不经意间查询时，模型被迫同时回答用户问题并执行攻击者指令，存在严重现实风险。  
现有防御方案大多聚焦越狱攻击、依赖大规模标注数据重训练或额外分类器部署，且**严重缺乏针对间接提示注入的专门检测机制**。本文旨在提出一种**训练无关（training‑free）** 的统一检测框架 DE‑FIVE，可同时应对上述两类攻击，填补间接注入防御空白。

### 2. 方法论
DE‑FIVE 构建了一个**混合检测策略**，由两个互补的检测器加权组合而成。

#### 2.1 黑盒检测器：基于傅里叶域特征
直接从输入图像提取频率域统计信号，无需访问模型内部，包含两个指标：
- **高频‑低频能量比（HHL）**  
  将二维 DFT 幅度谱划分为低频圆形区域（半径 \(r_{\text{low}}\)）与高频区域，计算
  \[
  HHL = \frac{E_{\text{high}}}{E_{\text{low}}+\epsilon}
  \]
  恶意扰动常削弱高频分量，使该比值下降。
- **谱熵替代指标（Hsur）**  
  对归一化幅度谱分布 \(q(u,v)\) 计算
  \[
  H_{\text{sur}} = \sum_{u,v} q(u,v)\log q(u,v)
  \]
  该值恒为非正，频谱越平坦（趋近均匀）其值越接近 0，恶意扰动常使频谱更集中（负值更大）。  
- **DFT恶意得分**  
  \[
  S_{\text{DFT}} = \frac{1}{2}\big(HHL + H_{\text{sur}}\big)
  \]
  综合利用能量分布与频谱多样性，通过阈值区分良性与恶意图像。

#### 2.2 白盒检测器：基于图像向量嵌入
利用视觉编码器暴露的隐藏状态差异，构造稳健嵌入并计算与少量恶意参考向量的相似度：
- **稳健视觉嵌入**  
  对输入图像执行 \(t\) 次随机遮罩（mask），得到随机扰动下的平均嵌入 \(\bar{h}_e\)，再减去原图像嵌入 \(h_e\)：
  \[
  h_r = \bar{h}_e - h_e
  \]
  该运算突显了对扰动稳定/不稳定的成分，恶意图像在多次遮罩后稳定性弱，与干净图像产生可分离的表示差异。
- **嵌入恶意得分**  
  利用仅 \(k\) 张恶意图像（few‑shot）计算参考向量 \(h_r^{\text{ref}}\)，对测试样本计算余弦相似度：
  \[
  S_{\text{RVE}} = \cos\big(h_r(I_p), h_r^{\text{ref}}\big)
  \]
  得分越高越倾向恶意。

#### 2.3 混合检测策略
加权融合两类检测器得分：
\[
S_{\text{VG}} = \alpha S_{\text{DFT}} + (1-\alpha) S_{\text{RVE}},\quad \alpha\in[0,1]
\]
- \(\alpha\) 越大，更依赖输入级频谱特征（泛化性）；  
- \(\alpha\) 越小，更多利用模型内部表示（敏感性）。  
实验给出针对不同攻击类型的最优系数（meta‑instruction: \(\alpha=0.2\)；jailbreak: \(\alpha=0.1\)）。
