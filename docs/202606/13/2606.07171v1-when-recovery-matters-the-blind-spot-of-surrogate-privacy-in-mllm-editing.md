## 摘要
多模态大语言模型（MLLMs）支持灵活的指令驱动图像编辑，但当用户图像暴露多样且用户特定的隐私内容时，隐私风险随之而来。典型的隐私保护策略通常在云端编辑前用替代内容替换敏感区域。然而，最终输出往往是编辑后的替代图像，而非期望的编辑后源图像，这在设计和评估中忽略了局部恢复。为此，我们提出SPPE（基于替代的隐私保护编辑），首个面向恢复的基准测试，涵盖36个细粒度隐私类别和65条编辑指令。它定义了两种互补任务：1）可编辑性评估，即在云端交互前估计替代图像能否产生与原始图像一致的编辑；2）替代到源的编辑恢复，评估编辑后的替代图像能否在保留编辑效果的同时转移回隐私源图像。我们为每个任务设计专用方法：ERMA通过指令感知的多模态关系建模预测替代可编辑性，而C2E-S2SER利用替代编辑对作为视觉编辑证据，以源图像为源保留锚点，实现循环一致恢复。在SPPE和InstructPix2Pix上的实验表明两项任务均取得一致改进。对于可编辑性评估，ERMA在SRCC和PLCC上分别比最佳基线提升13.9%和12.3%。对于替代到源的编辑恢复，C2E-S2SER在SPPE的全部8项源完整性和编辑一致性指标上均超越SOER。

## Abstract
Multimodal Large Language Models (MLLMs) enable flexible instruction-driven image editing, but privacy risks arise when user images expose diverse and user-specific private content. Canonical privacy protection strategies typically substitute sensitive regions with surrogate content before cloud editing. Yet, the resulting output is often an edited surrogate rather than the desired edited source image, neglecting the local recovery in both design and evaluation scope. To this end, we introduce SPPE (Surrogate-based Privacy-Preserving Editing), the first recovery-oriented benchmark covering 36 fine-grained privacy categories and 65 editing instructions. It defines two complementary tasks: 1) editability assessment, which estimates before cloud interaction whether a surrogate can induce an edit consistent with the original image; and 2) surrogate-to-source edit recovery, which evaluates whether the edited surrogate can be transferred back to the private source with the edit effect preserved. We address each task with a dedicated method: ERMA predicts surrogate editability through instruction-aware multimodal relation modeling, while \method performs cycle-consistent recovery by using the surrogate editing pair as visual edit evidence and the source image as a source-preserving anchor. Experiments on SPPE and InstructPix2Pix show consistent improvements on both tasks. For editability assessment, ERMA improves over the best-performing baselines by 13.9% in SRCC and 12.3% in PLCC. For surrogate-to-source edit recovery, C2E-S2SER outperforms SOER across all 8 source integrity and edit consistency metrics on SPPE.

## 1. 论文的核心问题与研究背景

- **核心问题**：多模态大语言模型（MLLMs）在云端进行指令驱动的图像编辑时，用户上传的图像可能包含人脸、证件、私密环境等敏感信息，存在隐私泄露风险。现有的主流保护策略是**本地生成替代图像（surrogate）**，将敏感区域替换为合成内容后再上传云端编辑。然而，云端返回的是**编辑后的替代图像，而非用户最终想要的编辑后的原始图像**。这一“输出侧”的恢复问题被现有研究普遍忽视，导致用户无法获得既保护隐私又完成编辑的源图。
- **研究背景**：已有隐私研究多聚焦于输入侧保护（如掩码、扰动），却未考虑编辑完成后如何将云端施加的编辑效果**无损地转移回私有源图**。文本占位符可以简单替换复原，但视觉替代无法直接替换，因为编辑效果是施加在替代区域上的，恢复必须同时保留源图身份细节并传递编辑效果。这一“恢复盲点”是本文要解决的核心挑战。

## 2. 方法论：核心思想、关键技术细节

论文提出了一个以恢复为导向的基准SPPE和两个分别应对可编辑性评估与编辑恢复的模型。

### 2.1 基准 SPPE 的双任务设计
- **任务1：可编辑性评估**（Editability Assessment）  
  在调用云端MLLM前，预测替代图像在给定编辑指令下能否产生与源图编辑方向一致的编辑效果。可编辑性定义为源图编辑方向 `I → I'` 与替代图编辑方向 `S → S'` 在CLIP语义空间中的余弦相似度。评估模型仅凭 `(I, S, P)` 预测该得分。
- **任务2：替代到源的编辑恢复**（Surrogate-to-Source Edit Recovery）  
  给定私有源图 `I`、替代图 `S`、编辑后替代 `S'`、隐私掩码 `M` 和提示 `P`，恢复编辑后的源图 `Î`。评估从源完整性（与 `I` 的相似度）和编辑一致性（与参考图 `I'` 的相似度和编辑方向对齐）两个维度进行。

### 2.2 ERMA：可编辑性评估模型
- **核心思想**：替代的可编辑性不仅取决于它与源图的视觉相似度，更取决于它是否保留了**指令相关的可编辑语义**。ERMA 通过多模态关系建模显式捕捉源图、替代图和编辑指令之间的语义关系。
- **技术细节**：
  - 使用冻结的CLIP编码器提取图像特征 `fI`，`fS` 和文本特征 `fT`。
  - 构建关系表示：`[fI; fS; fI ⊙ fS; fS - fI; |fS - fI|; fT; s(I,S); s(I,T); s(S,T)]`，其中包含元素积、差、相似度等线索。
  - 经过可训练的FFN块预测编辑性得分 `ŷ ∈ [0,1]`，用MSE损失优化。

### 2.3 C²E-S²SER：循环一致、编辑条件化的替代-源恢复模型
- **整体框架**：基于扩散Transformer（FLUX Inpainting），将恢复建模为条件生成：`Î = Rθ(I, S, S', M, P)`。
- **编辑条件化标签生成器**：从 `(S, S')` 预测编辑类型标签（添加/删除、编辑、风格），将其作为特殊token附加到文本提示前，使模型明确所需的编辑行为类型。
- **视觉条件构造**：将源图 `I`、替代 `S`、编辑后替代 `S'` 通过VAE编码后拼接为视觉条件 `CV = [zI || zS || zS']`，与文本token、噪声图像token一起输入DiT进行多模态自注意力。
- **循环一致正则化**：在训练的后半阶段引入反向恢复路径：用恢复结果 `Î` 和反向配对 `(S', S)` 重建原始源图 `I`，使正向编辑变换保持可逆性，减少非编辑区域的漂移。
- **优化**：总损失 = 正向流匹配损失 + 标签分类损失 + 循环一致性损失（后期加入）。采用LoRA高效微调，仅更新低秩适配参数。

## 3. 实验设计

- **数据集**：
  - **SPPE**：自建基准，含55,696个编辑实例，覆盖36个隐私类别（人脸、证件、财务等），65条多样化编辑指令。每个实例包含源图、掩码、替代图、编辑后替代图、编辑后源图参考。
  - **InstructPix2Pix**：公开数据集，用于泛化测试，包含未见图像和提示。

- **对比方法**：
  - **任务1（可编辑性评估）**：AFINE-FR, AHIQ, DISTS, ST-LPIPS, TOPIQ-FR（通用图像质量/感知相似度方法），均为源图-替代图评估，不具备指令感知。
  - **任务2（恢复）**：VISII（视觉指令反演），EditTransfer，Prompt-Diffusion，Cross-Image Attention（零样本外观迁移），SOER（替代导向恢复专门方法）。

- **评估指标**：
  - **任务1**：SRCC（秩相关系数），PLCC（线性相关系数）。
  - **任务2**：源完整性（CLIP-Sim, SSIM, PSNR vs. 源图 I），编辑一致性（CLIP-Sim, SSIM, PSNR vs. GT的I'），方向相似度 DirI 和 DirS。

## 6. 主要结论与发现

- 提出并强调了MLLM编辑中**替代隐私策略的恢复盲点**：现有方法只保护输入，忽略了输出端必须将编辑效果迁回源图。
- SPPE基准首次系统定义了可编辑性评估与替代-源恢复两个互补任务。
- ERMA通过指令感知的关系建模，在预测替代可编辑性上显著超越传统IQA度量，尤其在某些类别（如车牌）上拉开巨大差距。
- C²E-S²SER在所有8项恢复指标上均优于前沿的SOER，在源完整性与编辑一致性之间达到了最佳平衡，并展现良好的跨数据集泛化能力。
- 循环一致正则化和编辑条件化标签引导是提升恢复质量的关键组件。

## 8. 不足与局限

- **算力与训练细节缺失**：未说明硬件配置、训练用时等资源需求，可能影响复现难度评估。
