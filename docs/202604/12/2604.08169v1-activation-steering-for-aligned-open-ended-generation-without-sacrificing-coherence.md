## 摘要
大语言模型（LLM）的对齐比通常认为的更为脆弱：对抗性提示、良性微调、涌现性对齐失效以及目标泛化错误都可能触发失配行为。最近的证据表明，某些失配行为在激活空间中被编码为线性结构，这使得通过引导（steering）进行处理变得可行；而安全对齐已被证明主要控制前几个输出 token，导致后续生成过程处于无保护状态。这些发现促使激活引导成为一种轻量级的运行时防御手段，能够在整个生成过程中持续纠正失配的激活。我们评估了三种方法：应用均匀加性引导的固定系数引导（SwFC），以及两种新型的投影感知方法——目标投影引导（StTP）和镜像投影引导（StMP），后者利用逻辑回归决策边界，仅对激活值低于分布阈值的 token 进行选择性干预。我们使用恶意系统提示作为失配的可控代理，在两种威胁模型（不诚实和轻慢）以及两种架构（Llama-3.3-70B-Instruct, Qwen3-32B）下进行了评估。结果显示，所有方法都能在保持连贯性的同时，显著恢复目标特质（诚实和同情心）。此外，StTP 和 StMP 能更好地维持通用能力（MMLU、MT-Bench、AlpacaEval），并在多轮对话中产生更少的重复。

## Abstract
Alignment in LLMs is more brittle than commonly assumed: misalignment can be triggered by adversarial prompts, benign fine-tuning, emergent misalignment, and goal misgeneralization. Recent evidence suggests that some misalignment behaviors are encoded as linear structure in activation space, making it tractable via steering, while safety alignment has been shown to govern the first few output tokens primarily, leaving subsequent generation unguarded. These findings motivate activation steering as a lightweight runtime defense that continuously corrects misaligned activations throughout generation. We evaluate three methods: Steer-With-Fixed-Coeff (SwFC), which applies uniform additive steering, and two novel projection-aware methods, Steer-to-Target-Projection (StTP) and Steer-to-Mirror-Projection (StMP), that use a logistic regression decision boundary to selectively intervene only on tokens whose activations fall below distributional thresholds. Using malicious system prompts as a controlled proxy for misalignment, we evaluate under two threat models (dishonesty and dismissiveness) and two architectures (Llama-3.3-70B-Instruct, Qwen3-32B). All methods substantially recover target traits (honesty and compassion) while preserving coherence. StTP and StMP better maintain general capabilities (MMLU, MT-Bench, AlpacaEval) and produce less repetition in multi-turn conversations.

*   **对齐的脆弱性**：尽管 LLM 经过了安全对齐（SFT/RLHF），但这种对齐是“浅层”且易碎的。恶意提示、良性微调、涌现性失配或目标泛化错误都可能导致模型产生有害输出。
*   **安全对齐的局限性**：研究发现，现有的安全对齐往往只控制生成的前几个 token，随后的长文本生成过程往往处于无保护状态。
*   **线性表示假设**：失配行为在模型的激活空间中通常呈现线性结构。这为通过“激活引导”（在推理时修改内部表示）来持续纠正模型行为提供了理论基础。
*   **研究目标**：开发一种轻量级的运行时防御手段，在不牺牲文本连贯性和通用能力的前提下，实现对长文本生成的持续对齐修正。

### 2. 方法论
论文对比了三种激活引导方法，核心思想是利用逻辑回归（LR）在对比激活对（对齐 vs. 失配）上训练分类器，提取引导方向 $\hat{v}_\ell$ 和决策边界 $m_\ell$。

*   **SwFC (固定系数引导)**：最简单的方法，在指定层的所有 token 激活值上统一加上一个固定比例的引导向量。
*   **StTP (目标投影引导 - 创新点)**：一种投影感知方法。它只对那些投影值低于决策边界 $m_\ell$ 的 token 进行干预，将其投影值强制提升到目标分布的均值水平。
*   **StMP (镜像投影引导 - 创新点)**：另一种投影感知方法。对于低于边界的 token，将其激活值沿决策边界进行“镜像对称”反射，从而实现更自然的分布偏移。
*   **关键技术细节**：
    *   使用逻辑回归的权重向量作为引导方向。
    *   利用逻辑回归的偏置项自动确定干预阈值，无需额外超参数。
    *   支持“全 token 引导”（包括提示词）和“仅响应 token 引导”两种模式。

### 6. 主要结论与发现
*   **对齐恢复**：所有引导方法都能显著恢复被恶意提示压制的诚实和同情特质，使其接近甚至超过原始对齐模型的水平。
*   **选择性干预的优势**：StTP 和 StMP 在保持模型通用能力（MMLU 等）方面显著优于传统的 SwFC。
*   **解决重复问题**：在多轮对话中，传统的均匀引导（SwFC）会导致严重的文本重复，而投影感知方法（StTP/StMP）能有效抑制重复，保持生成的自然度。
*   **层级效应**：发现引导的最佳干预点通常在模型的中部层（Llama 为 23-34 层，Qwen 为 41-45 层）。

### 7. 优点（亮点）
*   **自适应性**：StTP 和 StMP 实现了 token 级别的选择性干预，只纠正“坏”的激活，保护“好”的激活。
*   **多维度验证**：通过交叉熵和嵌入距离等非 LLM 指标证明了行为改变源于表征空间的真实偏移，而非评价偏差。
*   **多轮对话关注**：首次系统性地研究了激活引导在长程对话中的稳定性和副作用（如重复率）。

### 8. 不足与局限
*   **白盒限制**：该方法需要访问模型的内部激活值，因此无法直接应用于闭源 API（如 GPT-4）。
*   **线性假设限制**：并非所有安全特征都是一维线性编码的，对于复杂的非线性失配可能效果有限。
*   **双重用途风险**：论文也指出，引导技术可能被恶意利用来诱导模型产生有害行为（尽管本文关注的是防御）。
*   **架构依赖**：最佳干预层在不同架构间差异较大，迁移到新模型时仍需进行层扫描。
